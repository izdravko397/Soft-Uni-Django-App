Welcome to the Library Management System

This Library Management System allows users to browse and manage a collection of books. Key features include:

  • Viewing available books with details such as title, author, and publication date.
  • Authenticated users can book available books and add comments.
  • Library managers have additional privileges to add, update, and delete books from the inventory.

To get started, please register or log in if you already have an account. Enjoy exploring our collection!
